# P0/P1/P2 Checklist

## P0: Cannot complete

- [ ] Screen purpose unclear.
- [ ] First meaningful action unclear.
- [ ] Current scope/location/selection unclear.
- [ ] Primary action icon-only or visually buried.
- [ ] Interactive elements do not look interactive.
- [ ] Loading/empty/no-results/error/disabled/permission states conflated.
- [ ] Error lacks cause, impact, or recovery.
- [ ] Disabled control lacks reason or enabling condition.
- [ ] Keyboard-only primary task fails.
- [ ] Focus invisible, inconsistent, or obscured.
- [ ] Accessible name/label missing for a control.
- [ ] Essential contrast or target size fails.
- [ ] State depends only on color, shape, location, animation, or memory.
- [ ] Destructive action lacks risk-appropriate prevention or recovery.
- [ ] State matrix missing.
- [ ] Novice simulation missing.
- [ ] Accessibility review missing.
- [ ] Counter-review missing.
- [ ] Verification claimed but not run.

## P1: Fix or explicitly defer

- [ ] Terminology inconsistent.
- [ ] Empty state weak but not blocking.
- [ ] Helper text insufficient.
- [ ] Visual density high.
- [ ] Secondary action competes with primary action.
- [ ] Responsive issue has workaround.
- [ ] Evidence incomplete for non-critical state.

## P2: Improvement

- [ ] Minor spacing or alignment issue.
- [ ] Microcopy can be clearer.
- [ ] Progressive disclosure can improve advanced options.
- [ ] Additional automation would reduce future risk.
