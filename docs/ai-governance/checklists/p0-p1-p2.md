# P0/P1/P2 Checklist

## P0: 完了不可

- [ ] screen purpose が不明確。
- [ ] first meaningful action が不明確。
- [ ] current scope/location/selection が不明確。
- [ ] primary action が icon-only または visually buried。
- [ ] interactive elements が interactive に見えない。
- [ ] loading/empty/no-results/error/disabled/permission state が混同されている。
- [ ] error に cause、impact、recovery がない。
- [ ] disabled control に reason または enabling condition がない。
- [ ] keyboard-only で primary task を完了できない。
- [ ] focus が見えない、一貫しない、または隠れている。
- [ ] control の accessible name/label がない。
- [ ] essential contrast または target size が基準を満たさない。
- [ ] state が color、shape、location、animation、memory のみに依存している。
- [ ] destructive action に risk に応じた prevention または recovery がない。
- [ ] state matrix がない。
- [ ] novice simulation がない。
- [ ] accessibility review がない。
- [ ] counter-review がない。
- [ ] 実行していない verification を実行済みとして主張している。

## P1: 修正または明示的延期

- [ ] terminology が一貫していない。
- [ ] empty state が弱いが blocking ではない。
- [ ] helper text が不十分。
- [ ] visual density が高い。
- [ ] secondary action が primary action と競合している。
- [ ] responsive issue に workaround はある。
- [ ] non-critical state の evidence が不完全。

## P2: 改善

- [ ] 軽微な spacing または alignment issue。
- [ ] microcopy をより明確にできる。
- [ ] progressive disclosure で advanced options を改善できる。
- [ ] 追加 automation で将来 risk を下げられる。
