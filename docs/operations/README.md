# 運用記録

このディレクトリは、本番・準本番で実際に観測した事象、ログ確認、暫定対応、恒久対応、残リスクを、PR本文やレビュー報告から独立して残すための場所です。

## 残すもの

- Cloud Run / Cloud Build / Firestore / Firebase Hosting など、運用環境で観測した障害・復旧・調査記録
- revision / instance 切替、デプロイ、設定変更など、再発時に時系列で追える事実
- 実施した確認、実施していない確認、根拠にしたPR・レポート・commit
- 再発防止として後続の開発者が守るべき運用上の学び

## 残さないもの

- 秘密情報、トークン、Cookie、認証ヘッダー、個人情報、ユーザー入力全文
- 未確認の推測を事実のように書いたもの
- 一時的な作業メモだけで、後から再利用できないもの
